## StudyTracker API

A robust, production-ready REST API built with TypeScript and Express to track study sessions, monitor academic subjects, and calculate analytical statistics. 

## 🚀 Live Demo
The API is securely deployed in the cloud and accessible at:
`https://studytracker-jd3x.onrender.com`

---

## 🛠️ Tech Stack & Architecture

- **Backend Platform:** Node.js, Express, TypeScript
- **Database:** MongoDB via Mongoose ODM
- **Authentication:** JSON Web Tokens (JWT) & Bcrypt hashing
- **Deployment & CI/CD:** Render (Web Services)
- **Cloud Database:** MongoDB Atlas Cluster

---

## 🔑 Features

- **User Authentication:** Secure registration and login endpoints utilizing JWT auth guards.
- **Subject Management:** Full CRUD operations to log academic subjects, customized with colors and tags.
- **Session Tracking:** Log focused study periods linked dynamically to individual subjects.
- **Cloud Analytics:** Relational aggregation engine pipelines to analyze total study durations.
